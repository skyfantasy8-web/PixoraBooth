# PixoraBooth Features

## Version 1.0

### Session
- Session Timer (5 Minutes)
- Countdown Before Every Photo
- Retake Current Photo
- Session Progress (1/4, 2/4, 3/4, 4/4)

### Camera
- Canon EOS 1100D Support
- Live View
- Auto Reconnect Camera
- Mirror Live View (On/Off)
- Camera Test

### Printing
- Epson Printer Support
- 4R Printing
- Print Copies
- Print Preview

### Files
- Save Original Photos
- Save Final Layout
- Auto Folder by Date

### Languages
- English (Default)
- Indonesian
- Thai

### Settings
- Camera Settings
- Printer Settings
- Language Settings
- Timer Settings

### Admin
- Login
- Dashboard
- Event History
- Error Log

### User Interface
- Fullscreen
- Dark Theme
- Modern Design
